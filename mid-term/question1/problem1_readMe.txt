
## How to run the code:
```terminal
python3 problem1_snowball.py < snowfall_input4.txt

```

## description
- read the file and create an array to store the numbers
- calculate total snow produced by the consecutive three days from the start to the end of the array recursively
- for each time, compare the snow created by these three days (A)  and the half of the total snow B
- if A is greater than B, then Yes
- else No
